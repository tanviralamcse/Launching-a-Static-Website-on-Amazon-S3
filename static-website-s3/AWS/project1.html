Project Summary

Project Name: Launching a Static Website on S3 with CI/CD

Project Description: This project involves setting up a static website hosted on Amazon S3, integrated with a CI/CD pipeline using GitHub Actions. The website automatically updates and deploys changes to the S3 bucket whenever changes are pushed to the GitHub repository.

AWS Services Used:

  • Amazon S3 (Static Website Hosting)
  • AWS IAM (User and Permissions Management)
  • Amazon CloudFront (CDN)
  • AWS Certificate Manager (SSL/TLS Certificates)
